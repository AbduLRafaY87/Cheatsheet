HTML CHEATSHEET
  1. HTML BOILER PLATE: This is a basic HTML boiler plate code
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="style.css">
    <title>Web Page Title</title>
</head>
<body>
    
</body>
</html>
  2. Ordered List: This is a basic HTML ordered list code
    <ol>
    <li>item # 1</li>
    <li>item # 2</li>
    <li>item # 3</li>
</ol>
  3. Unordered List: This is a basic HTML unordered list code
    <ul>
    <li>item # 1</li>
    <li>item # 2</li>
    <li>item # 3</li>
</ul>
  4. Div Element: This is a basic HTML div element code
    <div>element item</div>
  5. Div Element w/ Class: This is a basic HTML div element code with class
    <div class='class-name'>element item</div>
  6. Div Element w/ Id: This is a basic HTML div element code with id
    <div id='id-name'>element item</div>
  7. Heading Styles: These are basic HTML heading styles
    <h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
  8. Image Element: This is a basic HTML img element code
    <img src="image.png" alt="Dummy image">
  9. Bold Element: This is a basic HTML bold element code
    <b>Bold Text</b>
  10. Italic Element: This is a basic HTML italic element code
    <i>Italic Text</i>
  11. Underline Element: This is a basic HTML underline element code
    <u>Underlined Text</u>
  12. Horizontal Line: This is a basic HTML horizontal line code
    <hr>
  13. Line Break: This is a basic HTML line break code
    <br>
  14. Anchor Tag: This is a basic HTML anchor tag code
    <a href="index.html">HTML File</a>
  15. Anchor Tag In new Tab: This is a basic HTML anchor tag code (opens web page in new tab) 
    <a href="index.html" target='_blank'>HTML File</a>
